# ASO 서비스 명세서 (Phase 1)

> **브랜드**: blockbusterlab (인디게임닷컴 공식 파트너)
> **상태**: 확정 스펙 (2026-04-12 업데이트)

---

## 1. 서비스 개요

**ASO 분석 & 최적화 서비스** — 인디게임 개발자가 게임 정보와 원본 material을 전달하면, 장르별 Top 게임 벤치마크와 비교 분석하여 **바로 스토어에 적용 가능한 완성된 결과물**을 제작해 전달.

### 포지셔닝
- 인디게임닷컴의 K-인디 전문 미디어 지위 + 정무식/임재청의 게임 업계 네트워크 공신력
- 리포트/가이드가 아닌 **바로 적용 가능한 완성물** (텍스트 + 이미지)
- 한국어 기반 ASO 서비스 사실상 유일
- **마케팅 메시지에 "AI" 언급 없음** — 결과물 품질로 말한다

---

## 2. 입력 요구사항 (개발자가 전달할 것)

### 필수 항목

| 항목 | 상세 |
|------|------|
| **게임 스토어 링크** | Google Play URL (이미 런칭한 경우 필수) |
| **원본 스크린샷** | 최소 **5장**, **1920x1080 이상** 해상도 |
| **게임 로고** | 고해상도 원본 파일 |
| **핵심 특징 3가지** | 게임의 차별화 포인트 |
| **게임 장르** | 자동 분류 가능하지만 개발자 명시 권장 |
| **타겟 시장** | 한국/글로벌/일본/중국 등 |

### 선택 항목 (많을수록 좋음)

- 게임 트레일러/게임플레이 영상
- 캐릭터 아트/일러스트
- 게임 UI 에셋
- 개발사 소개 자료
- 수상 경력/미디어 노출 자료
- 경쟁작 참고 자료
- 특별히 강조하고 싶은 포인트
- 피해야 할 표현/방향성

## 3. 결과물 (Deliverables)

### 텍스트 결과물

| 항목 | 내용 |
|------|------|
| **게임 제목 대안** | 3개 (완성본, 바로 적용 가능) |
| **서브타이틀 대안** | 3개 (완성본) |
| **스토어 소개문구** | 완성본 (첫 252자 최적화) |
| **키워드 리스트** | 30~50개 (우선순위 포함) |
| **프로모션 해시태그** | SNS/커뮤니티용 추천 태그 |

### 이미지 결과물 ★ 핵심 차별점

**두 가지 형태로 동시 제공**:

#### A. 자동 생성 이미지 (주력)
- HTML/CSS 템플릿 기반 자동 렌더링
- Google Play 스크린샷 규격 맞춤
- 5~8장 PNG 파일
- 각 스크린샷에 ASO 키워드 반영된 오버레이 텍스트

#### B. 상세 제작 가이드 (보장)
- 슬롯별 상세 지침 (개발자가 직접 제작 원하는 경우)
- 각 슬롯: 원본 스크린샷 선택 권장 + 카피 완성본 + 폰트/색상/크기 + 레이아웃 와이어프레임 + 참고 예시
- 단계별 Canva 제작 가이드 (30분~1시간 내 직접 제작 가능 수준)

### 분석 결과물

| 항목 | 내용 |
|------|------|
| **장르별 Top 10 벤치마크** | 해당 장르 상위 게임 비교 분석 |
| **경쟁 게임 분석** | 유사 장르 5개 게임 집중 분석 |
| **ASO 점수** | 항목별 100점 만점 평가 |
| **개선 우선순위** | 효과 크기 기준 실행 순서 |
| **아이콘 피드백** | 장르 트렌드 대비 개선 방향 |

### (선택) 편집장 메시지
- 임재청 편집장의 환영/격려 메시지
- 게임에 대한 개인적 감상
- ASO 전문 조언이 아닌 관계성 메시지
- 생략 가능

## 4. 서비스 플로우

```
Day 0: 접수
├── 개발자가 웹사이트에서 신청 폼 작성
├── 원본 material 업로드
├── 결제 완료
└── 자동 이메일: "5영업일 내 결과물 전달" 확인

Day 1-2: 분석 & 결과물 생성
├── 스토어 데이터 자동 수집 (이미 런칭한 경우)
├── 장르별 벤치마크 비교 분석
├── 텍스트 결과물 자동 생성
├── 스크린샷 이미지 자동 렌더링 (HTML 템플릿 + Puppeteer)
├── 가이드 문서 자동 생성
└── 분석 리포트 자동 생성

Day 3-4: QC & 검수
├── 임재청이 결과물 확인 (오탈자/명백한 오류만)
├── (선택) 편집장 메시지 추가
└── 최종본 확정

Day 5: 전달
├── 고객 대시보드에 결과물 업로드
├── 이메일 알림
└── 압축 파일 다운로드
```

### 수정 정책
- 14일 내 **2회까지 무료 수정**
- 3회차부터는 추가 비용
- 수정 범위: 텍스트 조정 / 스크린샷 레이아웃 수정 / 오탈자

## 5. 스크린샷 제작 파이프라인 (확정)

### 전체 흐름

```
1. 원본 material 분석
    ├── 게임 장르 자동 분류
    ├── 원본 스크린샷 품질 평가
    └── 비주얼 아이덴티티 추출

2. 장르별 템플릿 선택
    └── 장르별로 사전 제작된 HTML/CSS 템플릿 10~15개 중 매칭

3. 카피 & 레이아웃 자동 생성
    ├── 각 스크린샷 슬롯별 메시지 생성
    ├── 키워드 포함 오버레이 텍스트
    └── 이미지 크롭/배치 결정

4. Puppeteer 자동 렌더링
    ├── HTML 템플릿 + 변수 = 최종 HTML
    ├── 헤드리스 브라우저로 렌더링
    └── PNG 파일 출력

5. 규격별 Export
    └── Google Play 스크린샷 규격 맞춤
```

### 가이드 문서 동시 생성

자동 렌더링과 동시에 가이드 문서도 자동 생성되어 제공됨.

고객이 선택:
- 자동 생성 이미지를 그대로 사용
- 또는 가이드 따라 Canva에서 직접 제작
- 또는 둘 다 활용 (기본 렌더링 + 커스텀 수정)

## 8. 서비스 운영 체계

### AI 시스템이 하는 일 (고객/마케팅에서는 언급 안 함)

- 스토어 데이터 수집
- 장르 분류 + 벤치마크 비교
- 텍스트 결과물 생성 (제목/서브타이틀/소개/키워드)
- HTML 템플릿 선택 + 변수 채우기
- 가이드 문서 생성

### 임재청 편집장이 하는 일

- 주문 접수 알림 수신
- 결과물 형식적 QC (오탈자, 명백한 오류)
- (선택) 편집장 메시지 작성
- [고객에게 전달] 버튼 클릭

**주문당 소요 시간: 10~15분**

### 정무식 발행인이 하는 일

- 주 2~3회 대시보드 확인
- 중요 고객 상담 (대형 스튜디오 등)
- Founding Partner 제안
- 월간 성과 확인

## 패키지별 배포 매체 수 제안 (Phase 2 서비스 설계용)

### Starter: 국내 집중 — 15개 매체

### Growth: 글로벌 기본 — 40~50개 매체
- 국내 Tier 1+2 (14) + 영어권 Tier 1+2 (20) + 일본 3 + 대만 2 + 영어권 인디 전문 5~10

### Premium: 풀 글로벌 — 70~100개 매체
- 전체 국내 + 전체 영어권 + 일본/중국어권 풀 + 유럽 다국어 + 모바일/콘솔 특화

## 매체 이메일 자동 배포 시스템 설계 참고사항

### 공개 이메일이 확인된 매체 (즉시 자동 배포 가능)
국내 8개 + 영어권 10개 + 일본 1개 + 중국어권 1개 + 기타 2개 = **약 22개**

### 사이트 내 폼 제출 필요
국내 15개 + 해외 40개 = **약 55개**
→ 자동 제출 봇 또는 편집장 수동 제출 필요

### 기자 개별 연락 필요 (고가치)
영어권 Tier 1 대부분
→ 임재청 편집장 네트워크 활용 영역

### 실무 설계 제안
1. **이메일 배포**: 공개 이메일 매체는 BCC 일괄 발송 (22개)
2. **폼 제출**: 주요 매체는 수동, 나머지는 Playwright 자동화 고려
3. **개인화 피칭**: 10~15개 핵심 기자는 편집장 직접 연락

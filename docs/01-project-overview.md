# 프로젝트 개요

## 한 줄 정의

**인디게임 개발자를 위한 글로벌 진출 자동화 플랫폼**
— 보도자료 제작/배포, 콘텐츠 번역, ASO 최적화를 AI 기반으로 자동화

---

## 배경

### 인디게임닷컴 (indiegame.com)

- 2003년 설립, 2021년 글로벌 인디게임 미디어로 개편
- 운영사: (주)플레이앱스 / 발행인: 정무식 (가천대 게임영상학과 부교수)
- "K-인디게임 전문 미디어"라는 독보적 포지션
- 뉴스, 리뷰, 공모전/지원사업 정보, GameCollege(교육) 운영
- 다국어 지원 (한/영/일/중)

### 왜 이 서비스를 만드는가

1. **인디게임 시장의 고성장**: 2025년 $48.5B → 2031년 $108.3B (CAGR 14.32%)
2. **인디 개발자의 공통 과제**: 마케팅 예산 부족, ASO 전문지식 부재, 글로벌 진출 장벽
3. **미디어의 한계 → 서비스 사업 확장**: 뉴스 보도만으로는 수익화 한계, 개발자 직접 지원 서비스로 사업 다각화
4. **편집장의 전문성 활용**: 20년+ 게임 업계 경험을 AI와 결합하여 확장 가능한 서비스 구축
5. **도메인 자산 활용**: "indiegame.com" 제네릭 도메인의 글로벌 SEO 잠재력

## 서비스 구성

### 3대 핵심 서비스

```
┌─────────────────────────────────────────────────────┐
│                indiegame.com 플랫폼                  │
├─────────────┬─────────────────┬─────────────────────┤
│  보도자료    │     번역        │      ASO             │
│  제작/배포   │   현지화        │    최적화             │
├─────────────┼─────────────────┼─────────────────────┤
│ - 보도자료   │ - 게임 내 텍스트│ - 키워드 리서치      │
│   작성/편집  │ - 스토어 설명   │ - 제목/서브타이틀    │
│ - 200+ 매체 │ - 보도자료 번역 │ - 스크린샷/아이콘    │
│   배포      │ - 25+ 언어     │ - 소개문구 최적화    │
│ - 성과 리포트│ - LQA 검수     │ - A/B 테스트 제안    │
└─────────────┴─────────────────┴─────────────────────┘
```

### 서비스 간 연계

```
보도자료 작성 → 자동 번역(영/일/중) → 다국어 배포
                    ↓
            스토어 소개문구 번역 → ASO 키워드 최적화 반영
```

세 서비스가 독립적이면서도 연계되어, 번들 이용 시 시너지 효과가 발생하는 구조.

## 타겟 고객

### 1차 타겟: 한국 인디게임 개발자/스튜디오

- 1~5인 소규모 팀
- 첫 게임 또는 글로벌 진출 첫 시도
- 마케팅/PR 전담 인력 없음
- 예산: 프로젝트당 50만~300만원

### 2차 타겟: 아시아 인디게임 개발자

- 일본, 중국, 동남아 소규모 스튜디오
- 영어/한국어 마케팅 지원 필요

### 3차 타겟: 글로벌 인디게임 개발자

- 전 세계 인디 개발자
- indiegame.com 도메인의 글로벌 인지도 활용

## 비즈니스 모델

### 요금 구조

| 플랜 | 금액 | 대상 | 포함 서비스 |
|------|------|------|-----------|
| **Starter** | 49만원~ / 프로젝트 | 소규모 스튜디오 | 서비스 1개 기본 패키지 |
| **Growth** | 129만원~ / 프로젝트 | 글로벌 진출 준비 | 서비스 2~3개 번들 |
| **Enterprise** | 맞춤 견적 | 대규모/장기 | 전체 서비스 + 전담 매니저 |

### 추가 옵션

- 추가 언어 번역: 15만원~/언어
- 긴급 처리: +50%
- 보도자료 추가: 20만원~/건
- A/B 테스트 추가: 15만원~/회

### 수익 구조 전망

```
단기: 서비스 과금 (프로젝트 단위)
중기: 구독 모델 (월간 ASO 모니터링 + 정기 보도자료)
장기: SaaS 플랫폼 (셀프서비스 ASO 도구 + API)
```

## 차별화 포인트

| 기존 대안 | indiegame.com 서비스 |
|----------|---------------------|
| 일반 PR 대행사 | 게임 전문, 게임 매체 네트워크 보유 |
| 일반 번역 서비스 | 게임 용어/맥락 이해, LQA 포함 |
| ASO 컨설팅 회사 | 장르별 벤치마크 DB 기반 자동 분석 |
| 개발자 직접 수행 | AI 자동화로 시간/비용 절감, 전문가 검수 |

### 핵심 자산

1. **편집장 스타일 AI**: 20년+ 게임 업계 전문가의 편집 스타일을 학습한 AI
2. **장르별 ASO 벤치마크 DB**: 9개 장르, 80+ Top 게임 분석 데이터 (정기 업데이트)
3. **게임 매체 네트워크**: 200+ 국내외 게임 미디어 배포 채널
4. **indiegame.com 도메인**: 글로벌 인디게임 커뮤니티 접근성

## 비전

```
Phase 1: 서비스 대행 (사람 + AI)
    인디 개발자의 요청을 받아 편집장 + AI가 함께 처리

Phase 2: 반자동화 플랫폼
    개발자가 직접 정보 입력 → AI 초안 자동 생성 → 편집장 검수 → 결과 전달

Phase 3: 셀프서비스 SaaS
    개발자가 직접 ASO 분석, 보도자료 생성, 번역을 수행하는 플랫폼
    (편집장은 프리미엄 검수 서비스로 전환)
```

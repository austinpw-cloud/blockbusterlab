# ASO 지식체계 — 폴더 가이드

이 폴더는 ASO 자동화 서비스가 판단 근거로 사용하는 **내부 지식 문서**다. 외부 공개용이 아니라 서비스 개발·운영용이므로 최소 분량·실용 중심.

## 파일

- `knowledge.md` — ASO 지식 본체. 서비스가 쓰는 원칙·매트릭스·의사결정 규칙.
- `sources.md` — 인용한 외부 자료 인덱스 (갱신 시 찾아가기 위한 용도).
- `99-archived/` — 1차 정리 전 기존 문서 + 기록용 원자료.

## 규칙

- 핵심 주장에만 본문에 괄호로 `(Apple 공식 / 2026-04 확인)` 정도 출처 표시.
- 수치 인용은 출처·날짜 필수. 표본 없는 "대부분 / 100%" 단정 금지.
- 스토어 정책 변경·신규 리포트 공개 시 `knowledge.md` 갱신, 같이 `sources.md` 업데이트.

## 코드 연동

- `website/src/lib/aso/principles.ts` 가 `knowledge.md` 의 기계 판독용 상수(필드 한도·장르 기본값 등)를 담는다. 둘은 항상 동기화.

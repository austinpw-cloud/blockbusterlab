## 우선순위 원칙

1. **Phase 1 완성이 최우선** — AI 분석 없이는 진짜 서비스 아님
2. **관리자 UI가 AI보다 먼저** — 주문 받고 처리할 방법이 있어야 AI 결과물도 의미있음
3. **인증은 빨리** — `/api/admin/*` 무인증 URL 노출은 보안 리스크
4. **이메일은 AI보다 먼저** — 접수 확인 없으면 고객 경험 깨짐
5. **외주 디자이너 없음** — AI 자동 생성 + 가이드 fallback 조합 고수
